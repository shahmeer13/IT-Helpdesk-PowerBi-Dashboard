This dataset represents a simulated IT Helpdesk environment and was used to build the Power BI dashboard in this project.

---

## Purpose of the Dataset
The dataset is designed to replicate real-world IT support scenarios. It demonstrates how Power BI can be used to:

- Track helpdesk performance  
- Monitor workload distribution  
- Identify high-priority issues  
- Measure resolution effectiveness  

---

## Dataset Contents
Each record in the dataset represents a single support ticket.

### Key Fields

| Column                  | Description                                |
|------------------------|--------------------------------------------|
| Ticket_ID              | Unique ticket identifier                   |
| Date_Created           | Date the ticket was raised                 |
| Date_Resolved          | Date the ticket was completed              |
| Category               | Type of IT issue                           |
| Priority               | Low / Medium / High / Critical             |
| Status                 | Open or Resolved                           |
| Assigned_To            | Technician assigned                        |
| Resolution_Time_Hours  | Time taken to close the ticket             |

---

## Data Source
This dataset was manually created for learning and demonstration purposes.  
It is structured to mirror realistic corporate IT Helpdesk ticketing environments.

---

## How the Data is Used
This dataset powers the analytical visuals within the Power BI dashboard, including:

- KPI performance cards  
- Priority-level analysis  
- Issue category trends  
- Technician workload and performance  
- Resolution rate and resolution time metrics
